# Image sources

All imagery from [Unsplash](https://unsplash.com), used under the [Unsplash License](https://unsplash.com/license).

## URL scheme

The HTML uses direct `images.unsplash.com` URLs with sizing query params:

    https://images.unsplash.com/photo-XXXXXXXXXXXXX?w=1600&fm=webp&q=85

Sizes: city hero 1920w, tour/blog cards 1200w.
Dev team self-hosts AVIF/WebP/JPEG via `<picture>` during WordPress port.
